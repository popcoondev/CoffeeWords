# Firebase サービスアカウントキーの取得方法

テストデータ作成スクリプトを実行するには、Firebase Admin SDKのサービスアカウントキーが必要です。
以下の手順でサービスアカウントキーを取得してください。

## 1. Firebase Consoleにアクセス

https://console.firebase.google.com/ にアクセスし、プロジェクトを選択します。

## 2. プロジェクト設定へ移動

左側のナビゲーションメニューから「プロジェクトの設定」（⚙アイコン）をクリックします。

## 3. サービスアカウントタブを選択

上部のタブから「サービスアカウント」を選択します。

## 4. 新しい秘密鍵を生成

「新しい秘密鍵を生成」ボタンをクリックします。
秘密鍵の生成を確認するダイアログが表示されますので、「鍵を生成」をクリックします。

## 5. JSONファイルを保存

ダウンロードしたJSONファイルを、プロジェクトのルートディレクトリに `serviceAccountKey.json` という名前で保存します。

```
/Users/mn/development/CoffeeWords/serviceAccountKey.json
```

## 注意事項

- この秘密鍵はプロジェクトの管理者権限を持っています。安全に保管してください。
- このファイルは絶対にGitリポジトリにコミットしないでください。
- `.gitignore` ファイルに `serviceAccountKey.json` を追加することを推奨します。

## スクリプト実行方法

サービスアカウントキーを配置したら、以下のコマンドでテストデータ作成スクリプトを実行できます：

```bash
node scripts/create-test-data.js
```

管理者ユーザーを作成する場合は：

```bash
node scripts/create-admin-user.js
```
